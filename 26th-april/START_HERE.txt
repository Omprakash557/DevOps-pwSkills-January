⚡ QUICK START (10 MINUTES)
────────────────────────────────────────────────────────────────────────────

1. Install Dependencies
   $ pip install -r requirements.txt
   
2. Run Tests
   $ pytest test_user_manager.py -v
   Expected: 60+ tests PASSED

3. Generate Coverage
   $ coverage run -m pytest test_user_manager.py
   $ coverage xml
   Expected: coverage.xml created

4. Setup Token (in SonarQube UI)
   Go to: http://localhost:9000
   My Account → Security → Generate Tokens
   $ export SONAR_TOKEN="squ_xxx"

5. Run Analysis
   $ chmod +x run_complete_analysis.sh
   $ ./run_complete_analysis.sh

6. View Results
   Open: http://localhost:9000/dashboard?id=python_user_management_app

🔧 PREREQUISITES
────────────────────────────────────────────────────────────────────────────

Required:
  ✓ Python 3.9 or higher
  ✓ Java 11 or higher (for SonarQube)
  ✓ SonarQube server running (http://localhost:9000)
  ✓ pip (Python package manager)

Install via:
  $ pip install pytest pytest-cov coverage
  $ brew install sonar-scanner  (macOS)
  $ choco install sonar-scanner (Windows)
  $ sudo apt-get install sonar-scanner (Linux)

📋 QUICK COMMANDS
────────────────────────────────────────────────────────────────────────────

# Install all packages
pip install -r requirements.txt

# Run all tests
pytest test_user_manager.py -v

# Run specific test class
pytest test_user_manager.py::TestUserCreation -v

# Generate coverage
coverage run -m pytest test_user_manager.py
coverage xml

# Run SonarQube analysis
sonar-scanner -Dsonar.projectKey=python_user_management_app -Dsonar.login=$SONAR_TOKEN

# Or use automation script
./run_complete_analysis.sh

❓ TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────────

Problem: pytest: command not found
  Solution: pip install pytest

Problem: SonarQube not running
  Solution: cd sonarqube-x.x.x/bin/linux-x86-64 && ./sonar.sh console

Problem: Invalid token
  Solution: Generate new token in SonarQube UI

Problem: coverage.xml not found
  Solution: coverage run -m pytest && coverage xml

See SONARQUBE_RUN_GUIDE.md for more detailed troubleshooting.
